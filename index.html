Showcase

Youtube Iframe API

Minha implementação de uma biblioteca para Youtube Iframe API com funções extras e eventos adicionais, como: alternar e sincronizar vários reprodutores, evento onplaying para chamar funções baseadas no tempo de reprodução e muito mais.

"Draggables"

Uma biblioteca para transformar qualquer parte do DOM em drag e drop e adicionar interações como "grudar" em elementos, tocar um áudio, encontrar o menor caminho entre dois pontos etc. Cada interação tem seu kit de eventos únicos que são acionados de forma independente.

Componente de Audio

Um componente de áudio interativo, que pode ser criado em "runtime" com músicas derivadas de qualquer link, e pode ser posicionado dentro de qualquer elemento ou ficar invisível. Funciona muito bem em conjunto com uma biblioteca que adiciona comandos que disparam eventos facilmente no componente. Interessantíssimo para criar playlists e um sistema de streaming, mas só pensei nisso enquanto digitava.

Melhorias serão implementadas nesse componente na próxima iteração...

Edição de Imagem

Uma biblioteca para permitir editar propriedades da imagem via typescript e em runtime. Tem algumas features como obter a paleta das cores que mais se destacam na imagem e permitir que o usuário salve a imagem no formato e tamanho que desejar.

"Resizables"

Uma biblioteca para permitir que o usuário mude o tamanho de qualquer elemento do DOM de forma controlada. Interessante quando combinado com o "Draggables"
