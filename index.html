YouTube Video Player Library

This is my implementation of a library for the YouTube Iframe API. It includes extra functions and additional events such as: toggling and synchronizing multiple players, using the onplaying event to trigger functions based on playback time, and much more.

Draggable Elements Library

This is a library that allows you to make any part of a webpage draggable and droppable. It also adds interactions like snapping to elements, playing audio, finding the shortest path between two points, etc. Each interaction has its own set of unique events that are triggered independently.

Audio Component

An interactive audio component that can be created on-the-fly with music from any link. It can be positioned within any element or kept invisible. It works seamlessly with a library that adds commands to easily trigger events in the component. It's great for creating playlists and a streaming system. I just came up with this idea while typing.

Improvements will be implemented in this component in the next iteration...

Image Editing Library

This library enables you to edit image properties using TypeScript during runtime. It includes features like extracting the prominent color palette from the image and allowing users to save the image in their desired format and size.
